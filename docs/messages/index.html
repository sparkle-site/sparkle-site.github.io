致詞

校長、導師與各科指導老師的話

🎓
校長的話

每年的五月，是屬於建中科學班學生最耀眼的季節。有人被數學的邏輯吸引，有人對物理的奧祕著迷，有人從化學反應中看到秩序的美，也有人在生物與資訊的交會間探索未知的世界。

研究的起點，往往只是一個單純的念頭：「為什麼會這樣？」從提出問題、設計實驗，到不斷修正、重新思考，這條路或許辛苦，卻讓每一位同學都學會了什麼叫做「面對未知的勇氣」。這份勇氣正是學術研究最珍貴的精神。也許過程中曾有挫折與懷疑，但正是這些挑戰，讓你們更懂得如何堅持，也更相信自己。

無論未來你們走向哪個領域，這段經驗都會成為一份珍貴的養分。它教會你們觀察、思考與合作，也提醒你們 — 真正的科學家，不只是擁有知識的人，更是永遠對世界保持好奇、願意不斷學習的人。

我要特別感謝一路陪伴的校內的指導老師，還有大學與研究機構的專家學者，您們的熱情與用心，是學生前進的明燈。教育的力量，不在於給答案，而在於讓學生願意繼續提問、勇於探索。感謝您們讓這群年輕的夢想家，在學術的世界裡找到自己的方向。

好奇，讓我們開始；思考，讓我們成長。願每位同學都能帶著這份熱情與勇氣，在知識的旅途中繼續前行，讓思考成為你們一生最強大的力量！

導師的話

緣分讓我們相聚在 COS16 屆，每位同學就像是各據一方的星星，獨自閃耀，而我是星空下的觀星人，靜靜欣賞卻又不時燃起心中的點點花火……

相見之初，我們彼此觀望，我主動出擊試探，想進一步了解同學，有些同學個性溫暖給予我正向回饋，偶然間產生短暫微光的火花，點亮了老師心中的熱情蠟燭；接著有些同學開始釋放能量，展現在多元競賽中，如能力賽、奧賽或是體育競賽中皆有亮眼的成績，驚呼眾人，甚至在段考中遙遙領先的分數天花板，這所謂的「電」可真是將許多同學電得七葷八素、火花亂竄；課業之餘，你們也不缺席班際活動，如校慶園遊會、服務學習競賽、大隊接力、國際交流至今日的成果發表，籌備活動過程中，同學分工合作，有人不斷燃燒自己；當然也有人享受別人的照耀，最後活動總能圓滿結束，著實感謝那些不斷燃燒氧化自己放出燦爛光芒的同學，這是仙女棒級的火花～

今天，期許站在講台上的各位，將蓄積近一年半的專題研究能量能盡情地爆發出來，聲若洪鐘分享自己的努力，讓全場的鎂光燈聚焦在你身上。這個火花將會是絢麗、溫暖而難以忘懷。

各科指導老師的話

📐

數學

致 27 班的同學們

紅樓穿堂的粽子， 積水的走廊， 喧鬧教室內的鍾聲（鍾聲無誤）， 建中生活是場遊戲，我們都在遊戲中學習。

鋒芒初露，不過是實力而已，修課完、考試後，同學名次的排列數是幾？ 廣闊夜空的恆星閃耀，星體各自運行，看似獨立，卻被無形的力量牽引。 微觀世界的吸引排斥，共享一對電子，沉穩確實，逐漸構成萬物的根基。 達爾文之說彪炳，塵世之偶然，留下的不是最強，是最能適應變化的你。 你說未來會如何？ 我說，相信會如你所願的。

人生是一場冒險、一堆提問、一段與自己對話的過程。 希望大家可以接受世間的不完美， 但也感謝這一切的相遇。

物理

從小時候開始，相信各位對於科學就一直有閃閃發亮的好奇心，終於在這個年紀，有了更為深厚的理論基礎，而可以將感興趣的問題做一個較為深入的研究。現在辛苦了一年多，在這個不一樣的時刻，讓我們一起跟同學分享與欣賞物理專題的有趣之處！

🧪

化學

回望這一年半的歷程，從分組、寫電子郵件、尋找實驗室與聯繫教授、確立題目，到進入實驗室學習、與學長姐磨合；從提出假設、設計流程、反覆修正，到一次次面對失敗挫折、再站起來重做，你們以想像力、直覺、推理與觀察，走過科學研究最真實的日常。終於，數據逐漸成形，成果得以驗證；寫作、修訂、簡報與口頭發表，也讓思考更為精煉與順暢。站在前人的肩膀上，你們已踏入科學之門。未來在 AI 的輔助下，願你們更能專注於提出好問題、做嚴謹的實驗、養成求真的品格，在通往科學大道的路上，走得深遠而篤定。更願你們記得，科學不僅是技巧與工具，更是誠實、耐心與合作的實踐。期許你們在成發之後，帶著自信與謙遜，迎向更寬廣的探索舞台。建中以你們為榮，師長亦深以你們為傲。

—— 中華民國 115 年 4 月 27 日 21:40 于桃園 龍潭

🧬

生物

從專題科目選擇開始，這道「天擇」力量終將讓同學進行「分歧型演化」，選擇生專的同學終將保留任勞任怨、隨遇而安與動靜皆宜的性狀（笑～）

瑛太為了解不同植物種的親緣關係，選擇擴充性狀在生物資訊領域開外掛；祉樂鑽研在蛋白調控端粒酶活性的探討，在茫茫分生大海中試圖找尋可讓動物長生不老的可能性；庭溱試著在優化腦瘤治療藥物的研究中，滿足一絲對化學（真愛無誤）的期待；冠宇足跡遍布雙北廣篩細菌，期望能找抗黴菌的潛力化學物質；懷賦追隨富蘭克林女士經典的 X 光繞射實驗，期望解開 XCP1 蛋白質的結構；品鈞探討 IDOL 可透過代謝膽固醇影響免疫細胞活性，若解密成功也可能是營養學與免疫學上的 idol。

當然，說的容易，但在科學研究之路上大家總是跌跌撞撞，次次有 data，但張張不能用的辛酸誰能了解。不過，我想在每次實驗過程遇到挫折而後思考、除錯，反覆操作讓技巧純熟，提早見習實驗時的人際互動彷如小型社會縮影，最後學習表達與分享。這些終將成為未來茁壯你們的深厚養分，加油～

🌍

地科

親愛的科學班同學們

恭喜你們！在建中這趟充滿挑戰與鑽研科學的旅程中，各位已經完成了最重要的階段。回首這段日子，你們在實驗室與書本間付出的無數汗水，必然已化作豐碩的學養與深刻的回憶。能與大家相識並結緣，老師至感榮幸。

我常與大家分享科學史的演進，無非希望： 　• 學習省思批判：不盲從，保持對知識的渴望與質疑。 　• 堅持實事求是：永遠相信證據，嚴謹運用邏輯推理與演繹精神。 　• 不要迷信權威：我們看得遠是因為站在巨人的肩膀上，但巨人也會出錯，相信未來的你們一定能成為科學巨人，同樣需要保持謙虛自省。

當你們在各領域卓有所成時，務必心懷感恩，感謝家人的支持、師長的引導、同儕的切磋，以及社會給予的資源。飲水思源，回饋社會。

誠摯祝福大家：在未來的科學殿堂裡，光芒萬丈。

資訊

各位同學，大家好

恭喜各位迎來了專題成果發表會。這本成發手冊不僅紀錄了你們高中生涯的重要里程碑，更是兩年來探索未知的深刻痕跡。專題研究的核心價值，向來不拘泥於最終呈現的數據多麼完美無瑕，或系統多麼無懈可擊；更在於「學習如何做研究」的歷程。在沒有標準答案的題目中找尋方向，學習解決問題、獨立思考與批判的能力，這才是這堂課最珍貴的餽贈。

專題研究中，你們從題目發想開始，一步步靠著自己的努力推進。如何將腦海中天馬行空的點子，收斂成切實可行的計畫？在未知的留白與探索空間中，你們必須學會自己掌舵。這場對求知慾與自律的考驗需要時間與毅力去跨越，宛如一場漫長 Debug 的過程，而這份迷惘與摸索，正是培養研究者自主性的必經之路。

或許你們會察覺到，當初設定的理想藍圖與最終成果之間，存在著構想與實作上不得不的妥協。學習在有限的時間與精力梳理輕重緩急，並以坦然的姿態接納研究所呈現的真實樣貌，往往是比寫出完美程式碼更難能可貴的成長。專題成發不是終點而是一個階段性的 git commit，是你們學術生涯 main branch 上的一個節點。人生沒有 git revert 可以回到過去重來，我們只能不斷修正、前進。

這兩年我們在彼此的陪伴中共同成長，也感謝你們包容了一位同樣在摸索引導之道的初任教師，願你們帶著這段日子裡領悟到的思辨、自律，以及直面真實的韌性，在未來的學術殿堂與人生旅途中，繼續勇敢地探索，成為自己生命中最沉穩的領航員。
